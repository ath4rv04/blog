# StackFrame

Structured thinking for developers.

A modern development journal built with **Next.js** and **Convex** for writing, sharing, and exploring structured technical thoughts.

---

## Features

* Create and publish posts
* Full-text search
* Real-time comments
* Live presence
* Authentication
* Dark mode

---

## Tech

* Next.js (App Router)
* Convex
* TypeScript
* Tailwind CSS
* Shadcn/UI
* Zod + React Hook Form

---

## Setup

```bash
git clone https://github.com/ath4rv04/blog.git
cd blog
npm install
npx convex dev
npm run dev
```

Create `.env.local`:

```env
NEXT_PUBLIC_CONVEX_URL=your_convex_url
```
